This is a trainining project with next.js + typescript + tailwindcss. It contains a stack avatar using tailwindcss.

Technologies used:
 - Next.js 10.0.3
 - Typescript 4.1.2
 - Tailwind CSS 2.0.1

## Getting Started

## Install

```bash
yarn
```

## Start

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
